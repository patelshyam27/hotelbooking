# Hotel Booking Website 🏨

A simple PHP-based hotel booking system with account creation, login, room booking, profile view, and restaurant menu.

## 🔧 Features

- User registration and login
- View hotel details and rooms
- Book rooms with customer info
- Profile page for users
- Restaurant menu display
- Styled with CSS
- MySQL database connection

## 🚀 How to Run

1. Place project folder in `C:/xampp/htdocs/`
2. Start Apache and MySQL using XAMPP
3. Open phpMyAdmin and import the database
4. Access the site at: `http://localhost/hotel_booking_project`
